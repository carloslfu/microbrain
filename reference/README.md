# reference/ — the canon, pinned

Unmodified snapshots of Andrej Karpathy's two microgpt gists, vendored here
so the course can diff against ground truth. All rights to these files remain
with their author; they are included for study, with attribution and
gratitude. If you want to reuse the code itself, take it from the sources
below. (If you're Andrej and want these snapshots removed or handled
differently — say the word.)

**microgpt.py** — the finished artifact, fetched at its then-current revision
(199 lines) on 2026-07-21:
https://gist.github.com/karpathy/8627fe009c40f57531cb18360106ce95

**train0.py … train5.py** — the six *revisions* of his build_microgpt.py
gist, one file per revision, oldest to newest. Each revision is one rung of
the published ladder:
https://gist.github.com/karpathy/561ac2de12a47cc06a23691e1be9543a
